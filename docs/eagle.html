Eagle from Autodesk

Eagle is a free cad program that allows you to lay out printed-circuit-board (PCB)s.
PCBs handle all the wiring so all you need to do is solder in the components.
Eagle is available for download here for windows.
This project uses 3 pcbs that are combined in one layout:
  • Bluetooth Joystick pcb
  • Pi Hat pcb
  • Upper tank pcb (for infra-red transmit and receive, turret movement and pixels)
