Original Documentation

The RC vehicle project utilizes a variety of language files and hardware components: Language files:
  • .py (python) on raspberry pi
  • .sh (setup files) on raspberry pi
  • .ino (Arduino) c-like programs
  • .cpp (Arduino) library files
  • .sch (Eagle schematic) for pi hat and joystick
Hardware components (Joystick controlled model):
  • JDY-40 tx and rx
  • 2 joystick component bought from ebay
  • joystick using above components and battery
  • ESC motor controller for treads and turret (left/right, up down)
  • Intra-red tx, rx and pixel (health)
  • Camera for First Person Viewer (FPV)
  • 3.3V voltage regulator
Hardware components (Internet controlled model):
  • Raspberry pi displayed web-page
  • esp32 Cam
  • Controlled with commands to esp32 cam?
Reliability and lack of lag, and simplicity are the driving forces behind this design.
