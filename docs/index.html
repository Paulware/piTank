Treaded RC vehicle with joystick and camera

The RC vehicle project utilizes a variety of language files and hardware components: Language files:
  • .py (python) on raspberry pi
  • .sh (setup files) on raspberry pi
  • .ino (Arduino) c-like programs
  • .cpp (Arduino) library files
  • .sch (Eagle schematic) for pi hat and joystick
Hardware components:
  • Raspberry pi 4
  • Raspberry pi camera
  • pi hat
  • Arduino Nano
  • Vehicle top pcb
  • ESC motor controller
Reliability and lack of lag, and simplicity are the driving forces behind this design.
Concepts
  • raspberry pi setup
  • ESC motor controller
  • Infra-red
  • Heng Long
  • esp32 (alternate camera)
