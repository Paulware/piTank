Camera Setup

  • Power to 5V
  • Gnd to Gnd
  • TX to UOR
  • RX to UOT
  • Gnd to IO1
  • Disconnect and reconnect 5V (to put unit in flash-programming mode)

Arduino Setup

  • File, preferences, Additional Board Manager URLs:
  • https://dl.espressif.com/dl/package_esp32_index.json, http://arduino.esp8266.com/stable/package_esp8266com_index.json
  • Press Ok
  • Tools, Board, Boards Manager
  • type: esp32, search
  • Install Esp32 by espressif
  • Tools, Choose board esp32 dev module
  • Change // Select Camera Module to: #define CAMERA_MODEL_AI_THINKER
  • Set SSID and password for your wifi network
  • Change Tools=>Board to Esp32 WRover Module
  • Tools -> Partition Scheme:change "Default" to "Huge APP(3MB No OTA)"
  • File, Examples, Exp32, Camera, Camera Web Server
  • Sketch, Upload
  • Disconnect IO1 to GND (Taking the unit out of flash-programming mode)
  • Tools-> serial monitor (Baud rate = 115200)
  • Press reset button (under)
  • Sketch, Upload

Trouble Shooting

  • File too big => Change Partition scheme as described above
  • Does not connect => Swap UOR and UOT pins
  • Does not connect => Reset device
  • Nothing in serial monitor => Make sure baud rate is 9600
