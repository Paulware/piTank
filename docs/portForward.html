How to Port Forward (MediaCom)

  • Determine location of router (probably 192.168.0.1)


    To determine the location of the router run ipconfig:
    Hold the windows key and press the "r" key
    Enter cmd

    Enter ipconfig

    The address of the Default Gateway is the local ip address of your router

  • Login to the router config page


    At this point you will probably need to call Media Com. Have a recent copy of your bill because they will ask for your pin.
    Also if you are not 18, have your parents nearby, because Media Com will need their permission to continue
    Media will let you know your user name and password, stay on the line with them as you login

  • Media Com control panel and port forward




Raspberry pi setup is done by running the setup.sh file located in the setup directory:
  • Download the latest raspbian .img file from here
  • Copy the raspbian .img file to an sd card using win32diskimager
  • Copy setup/setup.sh from this github repo to the /boot directory on the pi sd card
  • Insert the sd card into the pi
  • Connect the raspberry pi to an hdmi terminal, mouse and keyboard
  • Power up the raspberry pi
  • Execute these commands from a line terminal on the pi
    • sudo bash
    • cd /boot
    • ./setup.sh
  • After the setup file completes perform these actions from the pi desktop:
    • Turn on wifi
    • Select wifi country
    • Pair with your bt keyboard (if applicable)
    • Turn on the joystick and pair with the joystick using bluetooth manager
