Command the Motor Driver

In this first test we will power on the pi using an external power supply, run the server, and command the motor driver
For this test you will need:
  1. Raspberry pi 4
  2. sd Card flashed with latest raspbian and setup for ssh/wifi
  3. Free software Putty and Winscp
  4. Adeept Hat
  5. Volt Meter
  6. 12-V power adapter
You will not need: A vehicle, or the camera

Some options

  1. Don't need a vehicle


    This first test we just want to make sure we can control the motor output, but we are also testing our power supply, and the wifi access point, as well as getting used to the software (putty and winscp).
  2. Harbor freight has a voltmeter for about $7.00.
    The motor output will positive and negative DC voltage (for forward and reverse). I built a little test rig that connected the motor output directly to the voltmeter. This is optional, but makes testing easier
