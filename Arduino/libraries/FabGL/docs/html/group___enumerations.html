◆ KbdMode

enum fabgl::KbdMode
strong

This enum defines how handle keyboard virtual keys.

Enumerator
NoVirtualKeys 

No virtual keys are generated

GenerateVirtualKeys 

Virtual keys are generated. You can call KeyboardClass.isVKDown() only.

CreateVirtualKeysQueue 

Virtual keys are generated and put on a queue. You can call KeyboardClass.isVKDown(), KeyboardClass.virtualKeyAvailable() and KeyboardClass.getNextVirtualKey()
